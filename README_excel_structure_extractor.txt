
[엑셀 구조 추출기 사용법]

이 프로그램은 원본 엑셀 파일을 네 PC에서만 읽고,
아래 정보만 텍스트/JSON 보고서로 저장합니다.

- 시트명
- 헤더 스캔
- 상위 몇 행 미리보기
- 수식 샘플
- VBA 모듈 코드(선택, 환경 따라 실패 가능)

==================================================
1. 설치
==================================================
터미널에서:

pip install openpyxl xlwings

※ VBA 추출을 안 쓸 거면 openpyxl만 있어도 됩니다.

==================================================
2. 실행
==================================================
python excel_structure_extractor.py

==================================================
3. 추천 사용 순서
==================================================
1) 파일 선택
2) 미리보기 값 마스킹 체크 유지
3) 보고서 생성
4) 생성된 *_structure_report.txt 파일 내용을 ChatGPT에 붙여넣기
5) "이 파일 기준으로 통째 교체용 VBA 다시 짜줘" 식으로 요청

==================================================
4. VBA 추출이 안 될 때
==================================================
- Windows + Excel 데스크톱 환경인지 확인
- xlwings 설치 확인
- Excel 옵션 > 보안 센터 > 매크로 설정 쪽에서
  "VBA 프로젝트 개체 모델에 대한 신뢰할 수 있는 액세스" 허용 필요 가능
- 실패해도 시트 구조 보고서는 생성됩니다

==================================================
5. 주의
==================================================
- 이 프로그램은 구조 파악용 도구입니다
- 민감정보가 걱정되면 미리보기 마스킹 유지 권장
- 실제 코드 수정은 ChatGPT에 보고서 내용을 보여주고 요청하는 흐름입니다
